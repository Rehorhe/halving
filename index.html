Как работает сайт

  • Данные по цене и графику берутся из публичного API CoinGecko и Binance (без ключей).
  • Цена обновляется автоматически каждые 20 секунд.
  • График обновляется при смене интервала, при этом сайт пробует сначала CoinGecko, а при ошибке — Binance.
  • Сайт чисто статический — можно бесплатно разместить на GitHub Pages, Netlify, Vercel и др.

# Bitcoin Live — статический сайт с графиком BTC

Простой одностраничный сайт, который:

- показывает **онлайн‑график BTC/USD**;
- отображает **текущую цену BTC** и изменение за 24 часа;
- считает **обратный отсчёт до следующего халвинга** Bitcoin;
- полностью статичен и подходит для бесплатного размещения на **GitHub Pages**, Netlify, Vercel и других хостингах статических сайтов.

## Технологии

- Чистый HTML/CSS/JavaScript (без сборщиков и фреймворков);
- [Chart.js](https://www.chartjs.org/) через CDN для графика;
- Публичный API [CoinGecko](https://www.coingecko.com/) (без ключа):
  - `GET /simple/price` — текущая цена и изменение за 24 часа;
  - `GET /coins/{id}/market_chart` — исторические данные для графика.

## Структура

- `index.html` — разметка страницы;
- `style.css` — стили (тёмная тема, карточки, адаптивная вёрстка);
- `script.js` — запросы к API, обновление графика и таймера;
- `README.md` — это описание.

## Запуск локально

Откройте `index.html` напрямую в браузере или поднимите простой HTTP‑сервер, например:

```bash
python -m http.server 8000
```

и перейдите по адресу `http://localhost:8000`.

## Размещение на GitHub Pages

1. Создайте новый репозиторий на GitHub.
2. Скопируйте файлы (`index.html`, `style.css`, `script.js`, `README.md`) в репозиторий.
3. Сделайте коммит и отправьте на GitHub.
4. В настройках репозитория откройте **Settings → Pages**.
5. В разделе **Branch** выберите, например, `main` и папку `/root`.
6. Сохраните — через некоторое время GitHub выдаст вам ссылку на сайт.

> Обратите внимание: даты халвинга в коде указаны ориентировочно, фактический момент события зависит от скорости добычи блоков.
